//prompt 2 (ข้อบังคับ) shallow Equality
ข้อนี้เป็นการเขียน function  shallowEquality เป็นการเปรียบเทียบว่า object2 ตัวนี้มี key และค่าเหมือน object1 ทุกอย่างเพราะปกติถ้าเราเขียนแบบเทียบ object ธรรมดาเครื่องหมาย === แบบนี้จะได้ค่า false เพราะถึงค่าข้างใน object จะเหมือนกันหมดเลยแต่มันเป็น object คนละตัวกันค่าก็ไม่เท่ากันแต่ข้อนี้ต้องการเทียบค่าข้างในให้มันเท่ากันเลยเขียน function Shallow Equality การเปรียบเทียบ object แบบเช็คค่าข้างใน โดยใช้ Object.keys() ดึงรายชื่อ object แต่ละตัวออกมาและคืนค่าเป็น array เริ่มด้วยการประกาศ function shallowEquality และรับ object 2 ตัวมาเปรียบเทียบ และประกาศตัวแปร key 2 ตัวและใช้  Object.keys() มาดึงค่าข้างในตัวแปรที่เราประกาศและคืนค่าเป็น array ออกมาต่อมาใช้ if เพื่อกำหนดเงื่อนไขว่า ถ้าจำนวน key1 and key2 ไม่เท่ากันให้ return false ทันทีและต่อมาก็ใช้ loop for of วนเช็คค่า key แต่ละค่า ใช้ if ใน loop บอกเงื่อนไขเช็ค key ใน object1 และ object2 ถ้าไม่เท่ากันให้ return false ทันทีวนเช็คจนครบ
ถ้าตัวไหนไม่เข้าเงื่อนไข(แปลว่าค่าเท่ากันหมด) ก็จะ return true 
และวิธีที่หนูลองใช้อีกวิธีนึง Object.key() + .every() คือใช้ Object.key() แปลง object เป็น array ของคู่ [key, value] เหมือนเดิมแต่จะต่างตรงที่จะใช้ .every() ที่เป็น medtod ของ array แต่เราทำการแปลง Object จาก Object.key() ไปเป็น array แล้วก็สามารถใช้ได้และ .every() จะทำงานแทน for loop  เพราะ .every() จะเช็คทุก element ใน array ว่ามันตรงตามเงื่อนไขที่กำหนดมั้ย เริ่มโค้ดก็จะประกาศ function รับ parameter มา 2 ตัว สร้างตัวแปร keys1 Object.keys(obj1) คืนค่า array ของชื่อ key ทั้งหมดใน obj1 และสร้างตัวแปร let keys2 = Object.keys(obj2); ทำงานเหมือนกับ key1  ต่อมาใช้ if เช็คจำนวน key ของ obj1 กับ obj2 ว่าเท่ากันมั้ย ถ้าไม่เท่ากันให้ return false ทันที ต่อมาจะเป็นการใช้ Object.key() และ.every() เรื่อมก็จะใช้  Object.key() แปลงค่า object ให้เป็น array และใช้.every() จะวนทุก element (แต่ละ element คือ [key, value]) และใช้ array destructuring แยกออกมาเป็นตัวแปร และใช้มันเช็คว่า obj2[key] เท่ากับ value มั้ย ถ้าทุกคู่ keyและvalue เท่ากัน .every() จะคืนค่าคืน true แต่ถ้าไม่เท่ากันจะคืนค่า false เพราะ .every() จะคืนค่าเป็น Boolean เราเลยไม่ต้องกำหนดให้มัน return true หรือ false
ข้อนี้ก็จะได้เรียนรู้เรื่องการเปรียบเทียบ object แบบเช็คค่า key ข้างในด้วยเพราะปกติจะเช็คแบบธรรมดาคือดูแค่ว่ามันอ้าง object เดียวกันมั้ยและได้รู้ว่า object.key()ใช้ยังไงและได้ฝึกการเขียน loop มากขึ้นและได้ลองใช้วิธีใหม่ๆใช้ medtod ใหม่ที่สามารถทำให้งานง่ายขึ้นเพราะแบบ code ที่ใช้ .every() จะสั้นและกระชับกว่าเหมาะกับการเช็คค่าแบบตรงๆแต่แบบของอาจารย์จะเขียนเป็ฯขั้นตอนมากกว่าและสามารถเพิ่ม logic ได้ง่ายคือสามารถเพิ่ม if อื่นๆใน loop ได้เลย
