Array และ String
length ใช้นับจำนวนสมาชิกใน array / ตัวอักษรใน string ได้ลองใช array method push() → เพิ่มค่าใหม่เข้าไปใน array ...spread → กระจายค่าออกมาเป็นสมาชิกย่อย เช่นstring → array ของตัวอักษรarray → กระจายสมาชิกเข้าไปใน array/object อื่น และมีเรื่อง Loopfor (i=0; i < ...; i++) → เหมาะเมื่ออยากใช้ index หรือปรับแต่งเงื่อนไขเองfor...of → ใช้ง่ายกว่า อ่านค่าทีละตัวโดยตรง ไม่ต้องใช้ indexใช้ loop คู่กับ .includes(), .length เพื่อตรวจสอบค่าได้  มี typeof และการเช็คค่าtypeof → เช็คประเภทของข้อมูล Array.isArray() → เช็คว่าเป็น array จริงหรือไม่ ได้เช็คค่า null / undefined ก่อนทำงาน Object และ Method ที่เกี่ยวข้อง Object.entries(obj) → แปลง object → array ของคู่ [key, value] Object.keys(obj) → แปลง object → array ของ key การเปลี่ยน key → uppercase ด้วย .toUpperCase() ฝึก shallow equality (การเปรียบเทียบ object ตาม key/value) ใช้ .every() แทน for loop → โค้ดสั้น กระชับ และการใช้ if, else if, elseเช็คเงื่อนไขหลายขั้นตอน เช่น การให้เกรด, การเปรียบเทียบค่าต่างๆ return ค่าเมื่อเข้าเงื่อนไข   ส่วนเรื่อง Function ได้ฝึกการเขียนฟังก์ชัน เช่น: sumArray → หาผลรวมของ array
containsNumber → เช็คว่ามีเลขนั้นๆ ใน array ไหม countPositive → นับตัวเลขที่มากกว่า 0 getFreqOfWords → นับความถี่ของคำ (แปลงเป็น lowercase + split + นับด้วย object) ฟังก์ชันเฉลี่ยเกรด → ใช้ typeof, if, return ค่า Class และการสืบทอด (OOP Basics) class Person → property name, method getName() class Student extends Person → ใช้ super() รับ name, เพิ่ม studentId, มี method introduce()class Animal + class Dog + class Cat → การสืบทอด เพิ่ม property/method ของ subclass และใช้ new เพื่อสร้าง object แล้วเรียก method
