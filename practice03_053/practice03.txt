//prompt 1 ( โจทย์ในห้องที่ทำไม่ได้ )
//Write a function getFreqOfWords(sentence) that returns an object with keys representing unique words in lowercase and values representing the frequency of occurrences of each word (ignore case) in the sentence.
//If the input string is null or undefined, return undefined.
ข้อนี้ให้นับความที่ของคำที่เจอถ้าเจอ 1 คำก็นับ 1 เจอ 2 คำก็นับ 2 เริ่มแรกเราจะเช็คว่า input ที่รับเข้าไปว่าเป็นค่า null หรือ undefined มั้ยถ้าเป็น ให้ return undefined ต่อมาแล้วจะประกาศตัวแปรใหม่เพื่อให้เก็บค่าที่เปลี่ยนแปลง เราจะเปลี่ยนข้อความที่เป็นตัวพิมพ์ใหญ่ให้เป็นพิมพ์เล็กทั้งหมด ก็จะใช้ .toLowerCase() ละใช้ .split เพื่อแยก string ด้วยช่องว่าง กลายเป็น array เพราะจะได้แยกคำออกมาเป็นคำๆ ต่อมาก็สร้าง object ว่างไว้เก็บผลลัพธ์(เก็บเป็น key กับ value) และใน for loop ก็จะวนทีละคำประกาศตัวแปรใน for loop แลเวให้ดึงคำออกมาต่อมาจะใช้ freq[word] = (freq[word] || 0) + 1; ตัวนี้จะบอกว่า freq จะดูว่ามี key "word" อยู่รึยัง ถ้ายังปกติมันจะบอกว่า undefined หรือเรายังไม่ได้กำหนดค่าให้มัน ให้ใช้ || 0 แปลว่าแทนค่าเป็น 0 แทนแล้วถ้าเจอค่อยบวก 1 
ขอนี้ได้ฝึกเขียน function ได้ฝึกตรวจสอบ เช็ค type ค่า null กับ undefined ด้วยและได้ใช้ method LowerCase() เปลี่ยนค่าจากตัวใหญ่เป็นตัวเล็ก และ .split ได้รู้ว่า .split ใส่ " " สามารถเปลี่ยนค่าจาก string เป็น  array ได้และได้รู้วิธีนับคำแบบ short-circuit ทำให้เขียนโค้ดได้สั้นลงด้วย
